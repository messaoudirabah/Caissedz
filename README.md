# CaisseDZ | AI-Powered Retail Domination 🛒💎

CaisseDZ is an AI-powered POS ecosystem designed for high-precision retail management. Built with PySide6, it features a cinematic HUD with integrated AI predictors for sales intelligence. It offers robust inventory control, multi-language support (RTL/LTR), secure sessions, and encrypted licensing for a next-generation retail experience.

## 🚀 Key Features

*   **AI Predictors**: Integrated sales intelligence and predictive analytics for smarter retail decision-making.
*   **Cinematic HUD**: A premium PySide6 interface with glassmorphism and high-fidelity visual feedback.
*   **Inventory Mastery**: Full lifecycle management of products, stock levels, and category hierarchies.
*   **Secure Ecosystem**: Multi-tier authentication, encrypted license activation, and comprehensive audit logs.
*   **Session Control**: Precise daily session management (Open/Close) with automated cash carry-over.
*   **Global Reach**: Dynamic translation engine with full Right-to-Left (RTL) support for multi-regional deployment.

## 🛠️ Technical Stack

*   **Core**: Python 3.x & PySide6 (Qt for Python)
*   **AI Engine**: Custom Predictors & Pre-trained Models
*   **Database**: SQLite (DB_v2 Optimized)
*   **Services**: Printer Integration, Auth System, License Core
*   **Packaging**: PyInstaller & Inno Setup

## 📦 Quick Start

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/caissedz.git
    cd caissedz
    ```

2.  **Install Core Dependencies**:
    ```bash
    pip install PySide6 sqlite3
    ```

3.  **Install AI Workload (Optional)**:
    ```bash
    pip install -r requirements_ai.txt
    ```

4.  **Launch Phase**:
    ```bash
    python main.py
    ```
